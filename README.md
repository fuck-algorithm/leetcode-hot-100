# LeetCode 热题 Hot 100

[English](README.en.md)

🎯 让天下没有难学的算法，把一天能理解的知识，缩短到一个小时！

在线演示：https://fuck-algorithm.github.io/leetcode-hot-100/

## ✨ 特性

- 🗺️ **学习路径** - 按算法类型划分的学习路径，从入门到精通
- 🎬 **动画演示** - 部分题目配有算法动画，直观理解算法过程
- 📊 **进度追踪** - 记录你的刷题进度，支持经验值和等级系统
- 🏆 **修仙系统** - 完成题目获取经验值，从练气期修炼到大罗金仙
- 🎁 **宝箱奖励** - 完成阶段目标解锁宝箱，获取额外经验值
- 🌐 **中英双语** - 支持中文和英文界面
